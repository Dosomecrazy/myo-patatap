# PatatapController
Myo controller for Patatap. Created at MHacks V by @jhil and @arimelenci.

## Setup

1. Slip on your Myo armband.
2. Head on over to http://www.patatap.com.
3. Use the Myo gestures to create sick beatz.

## Key

Fist =            Bass
Wave Right =      Glitch
Wave Left =       Snare
Spread Fingers =  Bell
